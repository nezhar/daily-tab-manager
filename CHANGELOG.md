# Changelog

All notable changes to this project are documented in this file.

## [1.2.0] - 2026-06-06

### Added
- Date range filter for saved tabs with Past 3 Months, Past 6 Months, Past 12 Months, All, and year-specific options.

## [1.1.0] - 2026-03-22

### Added
- Collection action to copy a saved collection as Markdown links.
- Toolbar action to paste Markdown links and import them as a new collection.
- Parsing/validation for pasted Markdown links, including basic URL safety filtering.

### Changed
- Updated documentation for Markdown copy/paste collection workflows.
